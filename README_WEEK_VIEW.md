# 周视图功能实现

## 🎯 功能概述

这是一个类似滴答清单的周视图功能实现，为用户提供直观的周计划管理界面。

## ✨ 主要特性

### 📅 周视图显示
- **7天垂直列布局**: 清晰展示一周的所有日期
- **今日高亮**: 自动识别并高亮显示当前日期
- **智能周标题**: 动态计算并显示当前周的时间范围
- **响应式设计**: 完美适配桌面和移动设备

### ✅ 任务管理
- **创建任务**: 快速添加新任务到指定日期
- **编辑任务**: 修改任务内容、日期、类型和优先级
- **完成任务**: 一键切换任务完成状态
- **删除任务**: 移除不需要的任务
- **拖拽移动**: 直观地将任务拖拽到不同日期

### 🎨 视觉设计
- **毛玻璃效果**: 现代化的视觉设计
- **颜色标识**: 任务类型和优先级的颜色区分
- **流畅动画**: 优雅的过渡和交互效果
- **图标支持**: 丰富的图标增强用户体验

### 🔧 技术实现
- **Vue 3**: 使用最新的Composition API
- **TypeScript**: 完整的类型安全
- **Tailwind CSS**: 响应式样式设计
- **PouchDB**: 本地数据存储
- **拖拽API**: 原生拖拽支持

## 🚀 快速开始

### 1. 访问周视图
```bash
# 启动应用后访问
http://localhost:3000/time
```

### 2. 切换到周视图
- 点击底部导航栏的"周"按钮
- 或使用左右箭头切换到周视图

### 3. 创建第一个任务
1. 点击右下角的"+"悬浮按钮
2. 输入任务内容
3. 选择日期和优先级
4. 点击"创建"按钮

## 💡 使用示例

### 创建每日计划
```typescript
// 今天的任务
const todayTasks = [
  { text: '完成项目报告', priority: 'high' },
  { text: '团队会议', priority: 'medium' },
  { text: '代码审查', priority: 'low' }
]
```

### 拖拽任务
1. 按住任务卡片
2. 拖动到目标日期
3. 松开鼠标完成移动

### 设置任务类型
1. 进入类型管理 (`/todo`)
2. 创建自定义类型
3. 在创建任务时选择类型

## 🎨 自定义配置

### 修改颜色主题
编辑 `WeekView.vue` 中的颜色变量：
```vue
<style scoped>
.task-item.high-priority {
  border-left-color: #ef4444; /* 红色 */
}
.task-item.has-type {
  border-left-color: #3b82f6; /* 蓝色 */
}
</style>
```

### 调整布局
修改组件中的布局参数：
```vue
<div class="day-column" style="flex: 0 0 280px;">
```
