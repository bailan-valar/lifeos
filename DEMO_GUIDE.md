# 周视图功能演示指南

## 🎬 快速演示

### 1. 启动应用
```bash
# 开发模式启动
npm run dev

# 访问应用
open http://localhost:3000
```

### 2. 导航到周视图
1. 点击左侧导航栏的"时间"选项
2. 在时间页面底部，点击"周"按钮
3. 进入周视图界面

### 3. 功能演示流程

#### 第一步：创建任务
```
1. 点击右下角的蓝色"+"悬浮按钮
2. 在弹出对话框中输入："完成项目文档"
3. 选择日期：今天
4. 选择优先级：高
5. 点击"创建"按钮
```

#### 第二步：更多任务创建
```
1. 点击明天日期列下的"添加任务"按钮
2. 创建任务："团队会议"，优先级：中
3. 创建任务："代码审查"，优先级：低
4. 创建任务："客户沟通"，优先级：高
```

#### 第三步：任务操作
```
1. 完成任务：点击任务左侧的复选框
2. 编辑任务：点击任务右侧的编辑按钮
3. 移动任务：拖拽任务到不同日期
4. 查看详情：点击任务跳转到笔记
```

#### 第四步：类型管理
```
1. 在类型管理中创建新类型："工作"
2. 选择颜色：蓝色
3. 选择图标：公文包图标
4. 在创建任务时选择该类型
```

## 🎯 核心功能展示

### 1. 界面布局
- **7天列布局**: 清晰展示一周的所有日期
- **今日高亮**: 当天日期用蓝色背景标识
- **任务卡片**: 每个任务显示为独立的卡片

### 2. 任务状态
- **未完成**: 默认状态，显示为普通卡片
- **已完成**: 灰色显示，文字有删除线
- **高优先级**: 红色左边框标识
- **有类型**: 对应类型的颜色边框

### 3. 交互特性
- **拖拽移动**: 按住任务可拖拽到其他日期
- **快速完成**: 点击复选框快速切换状态
- **悬浮按钮**: 右下角"+"按钮快速添加任务
- **今天导航**: 点击"今天"按钮回到当前周

## 📱 移动端演示

### 触摸操作
1. **滑动查看**: 左右滑动查看不同日期
2. **点击添加**: 点击日期列下的添加按钮
3. **长按拖拽**: 长按任务后拖拽到其他日期
4. **快速操作**: 点击复选框快速完成任务

### 响应式特性
- **单列布局**: 移动端显示为单列垂直布局
- **触摸优化**: 所有交互都针对触摸操作优化
- **自适应高度**: 根据内容自动调整高度
