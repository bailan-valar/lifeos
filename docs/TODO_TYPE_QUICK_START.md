# 待办类型管理 - 快速开始指南

## 🎯 功能概述

待办类型管理功能允许您创建和管理不同类型的待办任务，帮助您更好地组织和分类工作。

## 🚀 快速开始

### 1. 访问待办类型管理

有两种方式可以访问待办类型管理：

#### 方式A: 从待办页面进入
```
1. 打开应用
2. 进入"待办"页面
3. 点击右上角的"类型管理"按钮
```

#### 方式B: 直接访问管理页面
```
直接访问 /todo-types 页面
```

### 2. 创建第一个待办类型

1. 点击"创建新待办类型"按钮
2. 填写类型信息：
   - **名称**: 例如"工作"、"个人"、"学习"等
   - **图标**: 从预设图标中选择一个
   - **颜色**: 选择一个识别颜色
   - **描述**: (可选) 添加说明
3. 点击"保存"

### 3. 预设的图标选项

- ✓ 检查圆圈 - 默认选项
- ★ 星形圆圈 - 重要事项
- 🚩 旗帜 - 紧急任务

- ❤ 爱心圆圈 - 个人事项
- ⚠ 警告圆圈 - 需要注意
- ⚡ 闪电圆圈 - 紧急重要
- 📅 日历圆圈 - 日程安排
- 🎯 目标 - 目标任务
- 📄 文档 - 文档相关

### 4. 预设的颜色选项

- 🔵 蓝色 (#3b82f6) - 默认

- 🩷 粉色 (#ec4899) - 个人

## 🔧 管理操作

### 编辑待办类型
1. 在类型列表中找到要编辑的类型
2. 点击编辑图标 (✏️)
3. 修改需要更改的信息
4. 点击"保存"

### 删除待办类型
1. 在类型列表中找到要删除的类型
2. 点击删除图标 (🗑️)
3. 确认删除操作

**注意**: 删除待办类型不会影响已创建的待办任务。

## 💡 最佳实践

### 1. 命名规范
- 使用简短、清晰的名称
- 保持一致性（如都使用两个字或三个字）
- 避免特殊字符

### 2. 颜色选择
- 为不同类型的待办选择不同的颜色
- 保持颜色对比度，便于区分
- 考虑颜色的情感含义

### 3. 图标搭配
- 选择与类型含义相关的图标
- 保持图标风格一致
- 避免过于相似的图标

### 4. 描述填写
- 简洁明了地说明类型的用途
- 包含使用场景或注意事项
- 保持描述简短（建议不超过50字）

## 🎨 界面说明

### 类型管理界面
- **左侧**: 图标和基本信息
- **右侧**: 操作按钮（编辑、删除）
- **颜色标识**: 左边框颜色对应类型颜色

### 创建/编辑界面
- **表单字段**: 按重要性排序
- **图标选择**: 网格布局，10个选项
- **颜色选择**: 5x2网格，10种颜色
- **操作按钮**: 取消和保存

## 🔄 工作流程建议

```

```

## 🚨 常见问题

### Q: 删除类型后，相关待办会消失吗？
A: 不会。删除类型只会删除类型定义，不会影响已创建的待办任务。

### Q: 可以创建多少个类型？
A: 理论上没有限制，但建议控制在10个以内，便于管理。

### Q: 如何修改类型的显示顺序？
A: 目前类型按照创建顺序显示。未来版本将支持拖拽排序。

### Q: 可以使用自定义图标吗？
A: 目前仅支持预设图标。自定义图标功能将在未来版本中提供。
